# AI Pipeline — System Architecture

## Стратегическая основа

Исследование показало: **«AI slop» эпоха закончилась**. Новая стратегия:

1. **ИИ + человек** вместо полного авто-ИИ
2. **Мультирынок** (Tier-1 страны) вместо одного языка
3. **Чёткая экономика** (CPM × retention × трудозатраты)
4. **Управление вниманием** (хук 0–8 сек, удержание на CTV)

## Module 1 — Topic Engine

**Задача:** Находить и ранжировать темы с учётом CPM, конкуренции, формата.

### Источники данных
- YouTube Trends API
- Google Trends
- Reddit API (r/investing, r/technology, etc.)
- Twitter/X API
- Telegram каналы (парсинг)

### Выходные данные для каждой темы
```json
{
  "title": "How AI is Replacing Financial Advisors in 2026",
  "niche": "FINANCE",
  "channelType": "INTELLECTUAL",
  "contentFormat": "DEEP_ESSAY",
  "targetMarkets": ["US", "CA", "AU", "NO"],
  "estimatedCPM": 28.50,
  "trendingScore": 87,
  "competitionLevel": "MEDIUM",
  "hookIdeas": [
    "90% of financial advisors don't know this is coming...",
    "I lost $50K before I learned this about AI investing"
  ],
  "keywords": ["AI", "financial advisor", "investing", "2026"],
  "estimatedRetention": 65
}
```

### API
```
GET  /api/topics              — список с фильтрами
GET  /api/topics/:id          — детали
POST /api/topics              — создать
PATCH /api/topics/:id         — обновить
POST /api/topics/:id/approve  — одобрить
POST /api/topics/:id/reject   — отклонить
POST /api/topics/generate     — AI-генерация
```

## Module 2 — Script Engine (планируется)

**Задача:** LLM генерирует два формата скриптов из одной темы.

### Формат FUEL (30–90 сек)
```
[0–3 сек]  ХУК: сильная фраза + визуальный трюк
[3–20 сек] 2–3 ключевых факта (без воды)
[20–30 сек] Мини-CTA: подписка / переход на интеллектуальный канал
```

### Формат DEEP ESSAY (8–15 мин)
```
[0–8 сек]   ХУК: интрига, неожиданный факт, вопрос
[8–60 сек]  Краткий обзор всей темы (зачем смотреть дальше)
[1–3 мин]   Блок 1: Объяснение
[3–6 мин]   Блок 2: Примеры и кейсы
[6–9 мин]   Блок 3: Ответы на типичные возражения
[9–12 мин]  Блок 4: Практические шаги
[12–15 мин] CTA: подписка, плейлист, следующий ролик
```

### Правило хука (критически важно)
- Решение «смотреть/нет» — в первые **8 секунд**
- К 60-й секунде уходит **55% аудитории**
- **Хук = отдельный этап разработки**, не часть основного текста

## Module 3 — Voice Engine (планируется)

**Задача:** TTS и клонирование голоса для мультиязычного контента.

### Режимы
- **FUEL:** быстрый TTS (ElevenLabs/OpenAI TTS)
- **INTELLECTUAL:** клон голоса автора (постоянный «голос бренда»)
- **Локализация:** отдельные аудиодорожки EN/DE/ES/JA

### YouTube Multi-Audio
Один ролик → несколько языков → не размазывает просмотры по дублям → усиливает «вес» видео в алгоритме.

## Module 4 — Media Pipeline (планируется)

**Задача:** Аватары + монтаж + CTV-оптимизация.

### CTV Требования (45% watch time = телевизоры)
- Крупный текст (читаемый с 3 метров)
- Чистый звук и музыка
- Темп монтажа: без мельтешения, с визуальным разнообразием
- Горизонтальный формат 16:9 для YouTube основного канала

### Сервисы аватаров
- HeyGen (основной)
- D-ID (альтернатива)
- Synthesia (для корпоративного контента)

### Монтажный шаблон DEEP
```
[AVATAR: студия] → [СЛАЙД/ГРАФИК] → [AVATAR: крупный план] → [B-ROLL] → ...
```

## Module 5 — Analytics Engine (планируется)

**Задача:** Метрики YouTube → обратная связь для улучшения пайплайна.

### Ключевые метрики
| Метрика | Цель | Ниже нормы |
|---------|------|-------------|
| Retention @60 сек | >50% | <30% |
| Retention @8 сек | >80% | <60% |
| CTR обложки | >4% | <2% |
| RPM (Tier-1) | >$8 | <$3 |
| Комментарии/1K просмотров | >5 | <1 |

### Обратная связь
- Плохой retention @8 сек → слабый хук → переписать
- Низкий CPM → не Tier-1 аудитория → корректировать таргетинг
- Много комментариев с вопросом X → тема для нового ролика

---

## Module 6 — Community Manager (планируется)

**Задача:** AI-ответы в комментариях + сбор идей для новых тем.

### Функции
- Сканирование новых комментариев
- Генерация черновиков ответов в стиле канала
- Классификация: вопрос / идея / критика / спам
- FAQ → автоматически попадает в Topic Engine как новые темы
- Количество комментариев выросло на **+38%** — это важный сигнал алгоритму

## Экономика контента

### Себестоимость минуты видео
| Элемент | FUEL | INTELLECTUAL |
|---------|------|---------------|
| Сценарий | AI авто | AI + правка человека |
| Озвучка | TTS $0.01/мин | Клон голоса $0.05/мин |
| Монтаж | Авто-шаблон | Аватар + B-roll |
| Локализация | Субтитры | Дубляж (EN + DE + ES) |
| **Итого** | **~$5–15/ролик** | **~$50–150/ролик** |

### ROI-расчёт (пример)
```
Ролик INTELLECTUAL:
- Стоимость: $100
- Длина: 12 мин
- Целевая аудитория: US/AU/NO (CPM $28)
- Нужно просмотров для окупаемости: ~3600
- При 10K просмотров прибыль: $180 (RPM ~$18)
```

## Локализация: двухэтапный выход на рынки

### Этап 1 — Тест (дёшево)
- Субтитры на EN/DE/ES
- Локализованные заголовки и описания
- Смотрим: какие страны органически подтягиваются?

### Этап 2 — Масштаб (для показавших потенциал рынков)
- AI-дубляж с сохранением тембра (ElevenLabs Dubbing)
- YouTube Multi-Audio (один ролик, несколько озвучек)
- Локализованные обложки

## «Серые» схемы: политика

**TikTok-компиляции — НЕ основа бизнеса.**

Риски: страйки, отключение монетизации, потеря канала.

Возможная роль:
- Тест ниш и визуальной стилистики
- Временный денежный поток → реинвестировать в оригинальный контент
- Только с добавлением собственного комментария/аналитики
